Setting Up ChangeRoom Shopify App

Account Page

Step 1:

Please fill in all the account page details first. Start the process by clicking connect, to link your store to ChangeRoom. Next, enter in your contact information so customers can reach you if they have any product, shipping related questions. Finally add your display address which is the website url the public can find you with for example https://changeroom.io .

Step 2:

Enter in a short description about your brand or your sustainbitliy values. Add your logo using the upload button. These elements will be added to your personalized brand page on the app. Next add your return policy and fulfilliment times to give your customers some informaton about these proceses. Go to the top of the account page and press the save button so you don't lose all your changes. Before we can complete the account page, we need to head to the sizing table page, use the left navigation to do so.

Sizing Table Page

Step 3:

We need you to create sizing charts for the products that you add on the marketplace. This will help us with categorization and user sizing help. Add you first size chart by pressing start.

Step 4:

Create a name of for the size chart, select the gender and the category the size chart belongs to. If the category isn't there for the clothing you are looking , we currently do not have that clothing category on our app. You can send a request for the category to our email - changeroomofficial@gmail.com which we can try to add in the future.

Step 5:

Add the size chart sizes by seperating sizes with commas from smallest to biggest size in the first input box. Select your desired sizing unit. Then enter the sizing details of the required body measurments for the category of clothing you choose. For better sizing recommendation accuracy, add additional measurments you have for the product. When the add measurments button is pressed, a diagram also pops up visualizing the measurments we are looking for.

After completing this step and ensuring all the information in the chart is correct, go to the top of the page and press publish and then save in the top right corner. If you would like to save your progress or double check the infomration, just press save and you'll be able come back and edit/publish the chart.

Step 6:

Keep on adding size charts until you have covered all the products you would like to add to our marketplace. When you are finished making size charts, you are ready to start making products avaiable on the ChangeRoom marketplace. Head back to the account page using the left navigation.

Publishing

Step 7:

Scroll all the way down to publshing which is one of the most important parts of the app. This is where you can add products to the marketplace, start by pressing manage avaiability.

Step 8:

This is the bulk editor where you assign products to their respective size charts. Assign size charts and check mark all the products you would like to add to the marketplace then click save.

Step 9:

It is possible that after a couple minutes your publishing section will look something like this. You will need to go into the bulk editor and resolve these errors. Please see the section below for common publishing errors with the bulk editor. For reference all products that have been published succesfully may still take up to 48 hours to show up on the marketplace and get approved by our team.

Common Publishing Errors

Size table not found/ Size table is missing:

Please add a size table to the product from the sizing table column.

Size table not published:

Please go to your size table page and make sure the size chart is published. There should be a blue tag signifying that its published. If there isn't, click edit the size chart, then ensure the information in the size chart is correct then press publish and save.

Product size options not included in size chart options:

The size chart that has been linked to this product doesn't include all the sizes in the product. For example if your product size variants are Small, Medium, Large and XLarge but the size charts are S,M,L, the size chart does not include the XL size variant which is the issue. Or if you your product variants use a different sizing system for example 40,42,44 while your size chart is S,M,L that would not work either.

To fix this issue go check the products size variants and then either create a new size chart for this product which includes all the sizing variants or edit the size chart itself. If you do choose to edit the size chart, make sure you aren't about to cause the same issues for other products that have been attached to the size chart.

Description Missing, Featured Image and all other product related errors

Please go to the product and add the required information to ensure a proper user experience for your customers
